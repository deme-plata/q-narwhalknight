# Q-NarwhalKnight v0.0.9-beta Release Summary

## 📅 Release Date: October 23, 2025

## 🎯 Release Focus: Peer Discovery Fix

### 🐛 Critical Bug Fixed

**Peer Discovery Event Loop Termination**
- **Issue**: libp2p event loop was crashing on dial errors, preventing peer discovery
- **Root Cause**: `self.swarm.dial(addr)?` propagated errors instead of logging and continuing
- **Impact**: Nodes could not maintain connections with other peers
- **Fix**: Changed error handling to log dial failures while keeping event loop running
- **File**: `crates/q-network/src/unified_network_manager.rs:343-348`

**Before (Broken)**:
```rust
self.swarm.dial(addr)?;  // ❌ Crashes event loop on error
```

**After (Fixed)**:
```rust
if let Err(e) = self.swarm.dial(addr.clone()) {
    warn!("⚠️ Failed to dial peer {} at {}: {}", peer_id, addr, e);
} else {
    info!("📞 Dialing peer {} at {}...", peer_id, addr);
}
```
