Proof of concept. Production Quillon Graph mining uses a Genus-2 hyperelliptic-curve VDF, not SHA3 PoW. This miner uses SHA3-256 with a configurable target as a demonstration of the WebGPU pattern and a useful TPS test harness against /api/v1/mining/submit. Genus-2 VDF port to WebGPU is tracked as a follow-up effort (~6 weeks). See README.md in this directory for the technical details.
